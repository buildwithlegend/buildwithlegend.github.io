## 中文版

Markdown 是工程師最愛的寫作格式。

例如：

```
# 大標
## 小標
**粗體**
```

優點：

- 超快  
- 超乾淨  
- 超好讀  

---

## English Version

Markdown is the easiest way to write technical content.

Example:

```
# Heading
**Bold**
```

Fast, clean, and readable.
