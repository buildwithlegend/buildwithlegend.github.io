## 中文版

Git 是一套「版本控制工具」。

白話來說：  
👉 就像遊戲存檔。

你可以：

- 回到過去版本  
- 看誰改了什麼  
- 避免程式爆炸  

### 必學指令

```
git init
git add .
git commit
```

---

## English Version

Git is a **version control system**.

Think of it as a save point in a game.

You can:

- Go back in time  
- Track changes  
- Prevent disasters
